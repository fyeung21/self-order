# Self-Order

A convienient ordering web-app for dimsum restaurants.

### Installation

```bash
curl https://install.meteor.com/ | sh
```

### Run in /self-order

```bash
Meteor
```

## Technologies 

* Meteor
* MongoDB
* React
* Semantic UI React
