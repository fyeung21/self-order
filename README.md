# Self-Order

A convienient ordering web-app for dimsum restaurants.

### Installation

```bash
curl https://install.meteor.com/ | sh
```

### Run in /self-order

```bash
Meteor
```
### To get database from MongoDB, in /self-order

```bash
meteor mongo
db.menu.find()
```

## Technologies

- Meteor
- MongoDB
- React
- Semantic UI React

## Personal Learnings

- Set up MongoDB

- Call db using Meteor Methods

- Experiement with Semantic UI React library
